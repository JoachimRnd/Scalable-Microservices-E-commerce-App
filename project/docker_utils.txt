FRONTEND : 


docker build -t scapp-frontend .


docker run -d -p 3000:3000 scapp-frontend

docker stop CONTAINER_ID

BACKEND :

docker build -t scapp-auth .


docker run -d -p 3001:80 scapp-auth

With Couch-DB :

docker run -d -p 3001:80 --name auth-service --link users-db scapp-auth


Get the logs :

docker logs CONTAINER_ID


STORAGE :

docker build -t kv-storage-system .

docker run -d -p 3002:5984 --name users-db kv-storage-system

Instacart : 

docker build -t instacart-sample .
